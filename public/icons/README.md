# PWA Icons

Для корректной работы PWA необходимо создать иконки следующих размеров:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Как создать иконки

1. **Используйте онлайн-генератор:**
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/

2. **Создайте в Figma/Canva:**
   - Создайте дизайн 512x512px
   - Экспортируйте во все нужные размеры

3. **Рекомендации по дизайну:**
   - Используйте розовую цветовую гамму (#FF6B8A, #FFB3BA)
   - Иконка должна быть простой и узнаваемой
   - Оставьте отступы для maskable иконок (safe zone)
